# STEP
1. githubアカウントを作成
2. リポジトリを作成
3. readmeファイル, 簡易htmlファイルを作成
4. github.ioが見れるか確認
5. ローカルにsshでクローン
6. ローカル環境でサーバーを立てられるか確認
7. https://html5up.net/ から好きなHPの型をダウンロード
8. git clone したディレクトリにダウンロードしてきたディレクトリの中身をコピー
9. addしてcommitしてmainにpush
10. github.ioが変更されているか確認
